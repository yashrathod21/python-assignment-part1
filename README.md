# Student Grade Tracker

## Assignment — Part 1: Python Basics & Control Flow

A command-line **Student Grade Tracker** that manages student data, computes results, and provides a summary report — built using core Python concepts.

---

## Features

- **Data Parsing & Profile Cleaning** — Cleans raw student data: normalizes names to Title Case, converts roll numbers to integers, and parses comma-separated marks strings into lists.
- **Name Validation** — Checks that every word in a student's name contains only alphabetic characters and prints a validity status for each profile.
- **Marks Analysis** — Uses loops and conditionals to assign grade labels (A+, A, B, C, F) per subject, calculates totals, averages, highest and lowest scoring subjects, and supports interactive subject entry via a while loop.
- **Class Performance Summary** — Generates a formatted table showing each student's average and Pass/Fail status, along with class-wide statistics.
- **String Manipulation Utility** — Strips, converts, counts, replaces, and splits a student essay string using Python's built-in string methods.
